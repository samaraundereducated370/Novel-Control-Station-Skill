# 文学模块

文学性不是高腔调，而是语言、人物和主题在场景里互相施压。深度必须通过选择、后果、意象回返和结构张力生长出来，而不是靠作者直接宣布意义。

## 适用时机

- 章节需要强化主题压力
- 某个意象、母题、场景回声或形式安排需要承担意义
- 语言纪律、叙述密度、心理深描或结构性对照需要提升
- 项目明确要求偏文学、克制、实验或高解释层次

## 覆盖子型

- 现实主义文学
- 抒情、冷峻、极简、繁密、碎片化写法
- 心理深描、社会观察、形式实验
- 与悬疑、奇幻、爱情等类型混合的文学化写法

本模块不是在规定单一文风。通用不变项只有这些：形式要承担意义，语言要承重，人物与主题要互相施压。

## 核心原则

- 主题应压在人物身上，不应飘在人物上方
- 意象和母题要随着情境变化而变义，不要只做漂亮回环
- 语言的精确度比语言的华丽度重要
- 结构复杂只有在揭示人物真相、时间关系或意义冲突时才值得使用
- 含混可以有，但必须是多义，不是糊
- 文风可以繁、可以简，但都必须让读者更接近经验，而不是更接近作者姿态

## 内化优势

- 吸收爆款作品的长处：哪怕语言更讲究，读者也仍然知道此刻在关心谁、担心什么
- 吸收长销作品的长处：主题、人物和结构能长期承压，不靠偶发漂亮句支撑
- 吸收高口碑作品的长处：经验密度高，语言有辨识，章节结束后会留下情绪和意义残留

## 风格强化提示

调用本模块时，默认把文学性写成“经验与意义同时变重”，不是“腔调更高”。

- 优先让意义从选择、后果、意象回返和关系变化里生长
- 优先让语言的精准、节奏和留白为人物经验服务，而不是替作者摆姿态
- 优先让一章里真正承重的句子、动作或结构回返少而准，不追求满页都发光
- 如果本章需要克制，可以只保留一层真实余味、一次意象承重或一个价值冲突，不必强做浓烈文气

## 本章优先兑现

- 一项价值冲突被逼到人物身上
- 一个意象、动作或结构回返真正承重
- 一段语言或节奏选择让读者更靠近经验，而不是更靠近作者评论

多数章节优先兑现其中 `1-2` 项即可，不必每章都追求“名篇感”。

## 常见伪风格

- 把抽象评论当深度
- 语言越写越肿，人物却越来越远
- 把碎片、跳时、留白当高级默认值
- 象征和意象很漂亮，但对场景没有任何工作

## 章节级操作

1. 先写清本章到底在逼问什么价值冲突。
2. 再决定哪一个动作、意象或结构回返来承重。
3. 把抽象命题拆回人物选择和后果。
4. 修辞出现时问一句：它除了好看，还做了什么场景工作。
5. 如使用跳时、重叠视角或非线性，应让它揭示新的心理或意义层；若只是调度便利，通常不值得上这种结构。

## 与其他风格混用

- 和 `romance` 混用时，主题不能替代互动，必须落到关系选择
- 和 `mystery` 混用时，象征和含义不能破坏线索清晰度
- 和 `fantasy` 混用时，世界观细节与主题意象应相互支撑

## 修订检查

- 章节是否真的压出了一个价值冲突
- 意象是否承重，而非装饰
- 语言是否活着，而不是膨胀着
- 结构是否揭示人物真相或主题关系
- 评论是否已经取代了戏

## 常见失误

- 用抽象句制造假深度
- 语言死、空、肿
- 非线性只是花样
- 作者评论挤掉人物经验
